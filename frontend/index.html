Green Potential Model

This model quantifies urban greening capacity in Delft using three-stage methodology.

Key Objectives

  • Mitigate Modifiable Areal Unit Problem (MAUP) using hexagonal grids.
  • Identify physical restriction areas (buildings, water, infra).
  • Assess equity via the Tree Equality Index (TEI).

$$TEI = \frac{2}{m^{2}\overline{S}_{g}}\sum_{j=1}^{m}j(S_{g,(j)}-\overline{S}_{g})$$
